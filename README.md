# Verbos Irregulares — Inglés

Aplicación web interactiva para practicar verbos irregulares en inglés,
desarrollada para apoyar el aprendizaje de forma entretenida y progresiva.

## ¿Qué hace?

- Presenta 25 verbos irregulares en modo aleatorio
- Practica en ambas direcciones: Infinitivo → Past Simple y viceversa
- Muestra pistas letra por letra cuando la respuesta es incorrecta
- Lleva registro de aciertos y verbos aprendidos en tiempo real
- Muestra un resumen final con porcentaje de aciertos al terminar

## Tecnologías

- HTML5
- CSS3
- JavaScript (vanilla)

## Cómo usar

1. Clona o descarga el repositorio
2. Abre el archivo `index.html` en cualquier navegador
3. No requiere instalación ni conexión a internet
